## 👋 About Me

I build **AI systems that think and act autonomously**. My focus: production-grade agent runtimes, multi-model orchestration, and self-healing infrastructure.

Currently shipping **[Mohini](https://github.com/darshjme/mohini)** — an open-source Agent Operating System written in Rust. One binary. 104 skills. 40 channels. 188 models. Zero downtime.
